# Narrative Game System Character Builder (NGSCB)

##Purpose

This is a character builder for the "Narrative Game System" by [Venture Land Games](http://www.venturelandgames.com).

##Guiding Principle

>"9. Smart data structures and dumb code works a lot better than the other way around." - Eric S. Raymond, [The Cathedral and the Bazaar](http://www.catb.org)

I emphasized the tables and queries in NGCSB much more than the procedures. NGSCB is a database, first and foremost.
